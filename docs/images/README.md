# 📸 User Guide Images

This folder contains screenshots for the PokeFind User Guide.

## Required Screenshots

Place the following images in this folder to complete the user guide:

### 1. `pokemon-radar-main.png`
- **Description**: Main interface showing the Pokemon Radar view
- **Shows**: Top controls with search radius and location info
- **Screenshot**: First image you provided

### 2. `pokemon-map-view.png`  
- **Description**: Map view with Pokemon spawns visible
- **Shows**: Blue location marker, colored Pokemon markers, search radius circle
- **Screenshot**: Second image you provided

### 3. `pokemon-popup-gengar.png`
- **Description**: Pokemon details popup for Gengar (#94)
- **Shows**: Pokemon sprite, rarity badge (Rare/Orange), IV stats, spawn times, Get Directions button
- **Screenshot**: Fourth image you provided

### 4. `location-search-modal.png`
- **Description**: Location search modal interface
- **Shows**: Address search field, coordinate inputs (Latitude/Longitude), Search buttons
- **Screenshot**: Third image you provided

### 5. `pokemon-popup-tyranitar.png`
- **Description**: Pokemon details popup for Tyranitar (#248)
- **Shows**: Very Rare rarity badge, different IV stats example
- **Screenshot**: Fifth image you provided

## Image Guidelines

- **Format**: PNG or JPG
- **Recommended size**: 800-1200px width for clarity on GitHub
- **Naming**: Use the exact filenames listed above for automatic linking in USERGUIDE.md

## Adding Images

1. Save your screenshots with the exact names listed above
2. Place them in this `/docs/images/` folder
3. Commit and push to GitHub
4. Images will automatically display in the User Guide
